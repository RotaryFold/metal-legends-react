## About The Project
Metal Legends is a small React project built with Vite. It includes a Home page with featured bands rendered from a JavaScript data array, responsive layouts, and multiple routes.

<p align="right">(<a href="#readme-top">back to top</a>)</p>

## Main Page Description
The **Home** page includes:
- Parallax hero title
- Featured bands section rendered from an array of objects
- A gallery grid
- A music play button
- A modal that opens band details

- `src/data/*` (data arrays used as JSON-like content)

## Getting Started
1. Install dependencies:
   ```bash
   npm install
   ```
2. Run the project:
   ```bash
   npm run dev
   ```
3. Open:
   - http://localhost:5173
   - http://localhost:5173/home
